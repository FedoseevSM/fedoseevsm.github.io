KeyPolice — это не инструмент, а экосистема для фрилансеров: берите заказы на модерацию, масштабируйте навыки и зарабатывайте на Open Source.

🌴 Ссылка: [keypolice.github.io](https://keypolice.github.io)

## Наше решение

Мы разработали KPOL как полноценную Open Source биржу:

- Доска заказов: заказчики публикуют задачи с фильтрами по ключам, бюджетом и дедлайнами.
- Распределение для фрилансеров: автоматический матч по навыкам, например React и API; фрилансеры берут заказы.
- Инструменты: готовые шаблоны, админ-панель для трекинга, внешняя интеграция платежей от 499 руб за задачу.
- Open Source основа: полный код на GitHub для форка и кастомизации.

## Стек

Развертывание за 5 минут.

- React + Vite — фронт.
- Markdown — бэк.
- GitHub — хостинг.

## Ожидаемые результаты биржи через год

- Объем: 500+ задач распределено; средний дедлайн — 2 дня.
- Эффективность: 90% заказов выполнены фрилансерами из пула; снижение спама в чатах на 95%.
- Монетизация: без комиссии; SaaS-версия для агентств; платформа разработана децентрализованно разработчиками для разработчиков.
- Сообщество: 300+ фрилансеров.

## Публикация

Сегодня реализована базовая авторизация для admin логина. Любой может оставить задачу через простую форму — укажите ключевые навыки, бюджет и дедлайн.

## Как опубликовать задачу за 1 минуту

1. Перейдите на страницу площадки.
2. Заполните форму: описание задачи, стек, цена от 499 руб.
3. Задача мгновенно появляется на доске — фрилансеры увидят матч по навыкам и возьмут в работу.

Важно: в поле ссылка оставьте ссылку на контактную форму или сообщения в чате с вами.
