Динамичный каталог для тысяч позиций: фильтры, быстрый поиск, синхронизация данных из 1С-Битрикс, удобный интерфейс для пользователей и рост конверсии для бизнеса.

## Что это за решение

Это современный каталог продукции на **React 18, TypeScript и Vite**, интегрированный с **Bitrix (БУС) API**.  
Он подходит для интернет-магазинов, B2B-каталогов, производственных компаний и фриланс-проектов, где важно быстро находить товары, управлять ассортиментом и поддерживать актуальные цены и остатки.

Проект объединяет:

- иерархические категории;
- продвинутые фильтры свойств;
- быстрый поиск с автокомплитом;
- корзину с автообновлением;
- экспорт каталога в PDF;
- формы с reCAPTCHA и загрузкой файлов;
- PWA-режим для оффлайн-доступа;
- полностью адаптивный интерфейс на Tailwind CSS.

## Решение

Мы реализовали интеграцию через **REST API БУС**:

- автоматическую выгрузку каталога;
- обновление цен и остатков в реальном времени;
- мультифильтры по категориям, брендам и ценам;
- поиск с автодополнением;
- адаптивную верстку под десктоп и мобильные устройства;
- SEO-оптимизацию и ускоренную загрузку страниц.

В результате каталог становится не просто витриной, а полноценным рабочим инструментом для продаж и обслуживания клиентов.

## Безопасность

Сборка проекта выполняется в **изолированном Docker-контейнере** на отдельном сервере.  
Это снижает риск внешнего вмешательства в код и упрощает контроль над процессом поставки.

Дополнительно:

- исходники не редактируются напрямую после сборки;
- API-запросы защищаются с использованием SHA-хеширования;
- данные каталога и пользовательские сессии передаются в более защищённом формате.
