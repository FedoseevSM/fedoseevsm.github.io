## Постановка задачи

- Требовалось настроить отображение фото в карточке товара для торговых предложений.
- Изображения должны были меняться в зависимости от цвета товара.
- Пример желаемого результата был приложен во вложении.
- Работа выполнялась в рамках доработки интерфейса сайта на Bitrix.
- Заказчик предоставил доступ к Bitrix.
- По задаче возник спор, так как результат работы продавца не устроил покупателя.

## Процесс выполнения

- Были добавлены необходимые свойства для торговых предложений.
- Настроено отображение данных в карточке товара.
- Визуальная часть была доработана под поведение разных цветовых вариантов.
- Работа велась по согласованной задаче и с доступом к админке Bitrix.
- После выполнения возникло разногласие по качеству результата.

## Результат

- Необходимые свойства и отображение данных были добавлены.
- Отображение фото для разных цветов в торговых предложениях было настроено.
- Арбитраж рассмотрел спор и направил заказ в работу для последующей отмены.
- Заказ был отменён по инициативе сторон.
- Отмена заказа не повлияла на рейтинг.
