## Постановка задачи

- Необходимо было сделать страницу с фильтрами для отображения автомобилей.
- Также требовалась отдельная страница товара с подробной информацией по каждому автомобилю.
- В фильтрах должны были быть доступны основные параметры авто: марка, модель, год, привод, КПП, пробег, цвет, комплектация, количество владельцев и цена.
- На странице товара нужно было вывести описание, фото и номер телефона.
- Важно было организовать удобный поиск и просмотр объявлений в понятной структуре.

## Процесс выполнения

- Была собрана страница каталога автомобилей с набором фильтров по ключевым параметрам.
- Отдельно реализована карточка товара с полным набором информации по автомобилю.
- Структура страницы была настроена так, чтобы пользователь мог быстро отфильтровать подходящие варианты.
- В карточке товара были предусмотрены блоки для описания, фотографий и контактного номера телефона.
- Решение было выполнено в логике удобного автомобильного каталога с акцентом на навигацию и читаемость.

## Результат

- Страница с фильтрами для автомобилей успешно реализована.
- Отдельная страница товара работает корректно и содержит всю необходимую информацию.
- Пользователь может удобно искать авто по заданным параметрам и открывать подробную карточку.
- Проект выполнен успешно и в срок.
- Заказчик остался доволен результатом.
