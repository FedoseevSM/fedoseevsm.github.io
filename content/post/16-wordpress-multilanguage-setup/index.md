## Постановка задачи

- Необходимо было настроить выбор языков на сайте на WordPress.
- Требовалась мультиязычность для трёх языков: английского, казахского и русского.
- Структура URL должна была быть организована по типу отдельных языковых версий сайта.
- Кнопка переключения языка должна была отображаться в футере и хедере.
- Отдельно требовалось, чтобы переключатель можно было настраивать через Elementor или через админку.

## Процесс выполнения

- Была проведена первичная оценка задачи по мультиязычности на WordPress.
- Стало понятно, что для корректной реализации нужно учитывать структуру сайта, шаблон и способ вывода переключателя языков.
- Рассматривалась логика, при которой языковые версии будут доступны по отдельным URL.
- Также учитывался вариант гибкой настройки кнопки через Elementor или панель администратора.
- До начала работ было принято решение не запускать выполнение, чтобы избежать ошибок и лишних переделок.

## Результат

- Заказ был отменён до начала выполнения.
- Отмена произошла до старта работ и без негативных последствий.
- Заказчик получил честную оценку задачи по сложности и способу реализации.
- Коммуникация по проекту осталась корректной и профессиональной.
- Решение позволило не начинать работу, если условия были недостаточно определены.
