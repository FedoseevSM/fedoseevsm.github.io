## Постановка задачи

- Требовалось сделать карусель на сайте Bitrix с возможностью добавлять VK-видео из админки.
- Основной акцент был на удобном управлении контентом без необходимости править код вручную.
- Видео должны были подключаться через административную часть сайта.
- Важным условием было сохранить корректное отображение и удобную навигацию в карусели.
- Решение должно было быть совместимо с существующей структурой сайта на Bitrix.

## Процесс выполнения

- Была реализована логика карусели для вывода видео в нужном формате.
- Добавлена возможность подставлять VK-видео через админку, чтобы контент-менеджер мог управлять им самостоятельно.
- Вёрстка и поведение карусели были адаптированы под требования сайта.
- Для интеграции видео использовался сценарий, при котором ссылка на ролик сохраняется в админке и затем отображается в карусели.
- Решение было сделано так, чтобы его можно было поддерживать и обновлять без сложных доработок.

## Результат

- Карусель с VK-видео успешно внедрена на сайт Bitrix.
- Видео можно добавлять из админки без ручного редактирования шаблонов.
- Пользователю удобно просматривать видеоконтент в формате карусели.
- Решение работает в рамках текущей структуры сайта и не ломает интерфейс.
- Задача выполнена успешно и в срок.
