## Постановка задачи

- Необходимо было на Tilda сделать Zero Block, в котором активными должны были быть кнопки с правой части картинки.
- Также требовалось добавить кнопку «sos».
- Задача относилась к точечной доработке интерфейса внутри Tilda.
- Заказчик ожидал реализацию интерактивных элементов на уже готовом визуальном блоке.
- Отдельно важно было сохранить корректное отображение и работу кнопок.

## Процесс выполнения

- До начала выполнения была проведена первичная оценка задачи.
- По результатам оценки стало понятно, что информации недостаточно для уверенного старта работ.
- Было отмечено, что задача находится вне основной специализации по текущему профилю.
- Заказчику было предложено выбрать другого исполнителя, чтобы избежать ошибок и лишних правок.
- Работа по задаче не начиналась, чтобы не затягивать процесс и не создавать риски по качеству.

## Результат

- Заказ был отменён до начала выполнения.
- Отмена произошла по согласованию сторон.
- Отмена заказа не повлияла на рейтинг.
- Заказчик получил честную оценку ситуации до старта работ.
- Решение позволило избежать некачественной реализации и потери времени.
