## Постановка задачи

- Требовалось подготовить сайт к рекламному запуску.
- В качестве первого варианта рассматривалось добавление микроразметки Schema.org.
- В качестве второго варианта рассматривалась подготовка отдельного товарного фида для каталога.
- У заказчика уже был готов основной товарный фид, но для рекламы требовался другой формат.
- Начать можно было только с одного направления, если оно дало бы нужный результат.
- Заказ был отменён до начала выполнения работ из-за отсутствия корректной информации для старта.

## Процесс выполнения

- Было уточнено, что ранее задачу уже обсуждал коллега заказчика.
- Рассматривались два возможных направления: микроразметка и отдельный фид.
- Для начала нужно было выбрать только одно из них.
- Работы не были начаты, так как заказ был отменён до старта.
- Отмена произошла по согласованию сторон.

## Результат

- Заказ не был запущен в работу.
- Работы по микроразметке и фиду не выполнялись.
- Отмена не повлияла на рейтинг.
- Ситуация завершилась без конфликта и без последствий для стороны исполнителя.
- Заказ был закрыт до начала реализации.
